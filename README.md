# HELIOS

**H**igh-**E**fficiency **L**ibrary for **I**ntelligent **O**perations and **S**ystems

A modular Python framework for building high-performance data pipelines,
schedulers, and monitoring systems.

## Project Structure

```
helios/
├── cli/        Command-line interface
├── core/       Engine and task scheduler
├── io/         Streaming readers and writers
├── models/     Serialisable data models
└── utils/      Caching and metrics utilities
tests/          Unit test suite
```

## Quick Start

```bash
python -m helios run my_pipeline.yaml
python -m helios version
```

## Modules

| Module | Description |
|---|---|
| `helios.core.engine` | Central orchestration engine |
| `helios.core.scheduler` | Priority-queue task scheduler |
| `helios.io.reader` | Streaming file / byte-stream reader |
| `helios.io.writer` | Buffered and rotating file writer |
| `helios.models.base` | Base serialisable model |
| `helios.models.pipeline` | Pipeline configuration model |
| `helios.utils.cache` | Thread-safe LRU cache + memoize decorator |
| `helios.utils.metrics` | Counter, Histogram, and MetricsRegistry |
